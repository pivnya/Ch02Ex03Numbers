# Ch02Ex03Numbers
Simple **.NET console app** that outputs the number of bytes in memory that each of the following number types uses and the **minimum** and **maximum** values they can have: 
* sbyte;
* byte;
* short;
* ushort;
* int;
* uint;
* long;
* ulong;
* Int128;
* UInt128;
* Half;
* float;
* double;
* decimal.
